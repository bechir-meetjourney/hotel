# Project Overview

## Purpose

**Diyafah** (ضيافة) is a SaaS platform that provides ready-made website templates for hotels and furnished apartments in Saudi Arabia. The platform lets users browse templates by region, preview them live, and—through the setup flow—choose a template for their project.

## Main Features

- **Template gallery** – Browse templates with filters by region (e.g. Madinah, Makkah, Hijaz, Central, Sulamani).
- **Live preview** – Each template can be previewed at a dedicated URL before selection.
- **Multiple templates** – Currently: **Riyadh** (capital-inspired) and **Madina** (Madinah-inspired). Each is self-contained with its own layout, header, footer, and styling.
- **Responsive design** – All templates work across mobile, tablet, and desktop.
- **RTL / LTR** – Full support for Arabic (RTL) and English (LTR).
- **Dark / light mode** – Supported where applicable (e.g. Madina template with theme switcher).

## Tech Stack (Summary)

- **Backend**: Laravel 12
- **Frontend**: React 19, TypeScript, Inertia.js
- **Build**: Vite
- **Styling**: Tailwind CSS 4
